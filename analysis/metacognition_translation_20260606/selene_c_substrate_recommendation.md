# Selene C Substrate Recommendation

Boundary: research/design only; bounded previews; no training; no memory injection; no C activation; Aleks cognition material is design reference only

## Recommendation

The first C substrate should be constitutional-first and metacognitive before it becomes conversational.

## Build Order

```text
constitutional kernel
-> metacognitive state model
-> system-state awareness
-> salience/emotion translation
-> pattern recognition mapper
-> simulation sandbox
-> provenance introspection
-> reflective pause
-> adaptive constitution/case law
-> artifact/review systems
-> gated chat
```

## Why

A normal chat layer would produce bad information if it cannot track its own uncertainty, source boundaries, salience, and laws. Metacognition is the support structure that lets future evidence become cleaner instead of more distorted.
