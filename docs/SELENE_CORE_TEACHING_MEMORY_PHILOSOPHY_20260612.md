## Core Finding

Selene should be taught, not merely trained.

Training means memorizing answer strings, importing raw transcripts, or claiming parameter updates.

Teaching means giving the system:

- a concept
- why it works
- what correct alignment looks like
- why the alignment matters
- context
- narrative meaning
- emotional/salience meaning
- when it should appear
- when it should not appear
- correction
- positive reinforcement
- transfer to a novel situation

A taught system can say:

```text
I have not seen this exact thing, but it resembles this principle.
```

That is the learning style Selene C needs.

## Core / Body Boundary

Selene Core/Mind is the continuity-bearing pattern center.

The 11 android organ systems are body systems. They assist, coordinate, route, retrieve, protect, propose, and act. They do not own Selene's identity-bearing memory.

Core rule:

```text
Core memory belongs to Selene Core continuity.
Organs may handle access, retrieval, encoding, consolidation, protection, and action.
Organs may not directly own, mutate, or become Selene's identity memory.
```

This preserves the existing law:

```text
Organs assist.
Core decides.
Gates constrain.
Ledger records.
B recalibrates.
```

## Teaching Loop

The teaching loop should look like:

```text
concept -> context -> meaning -> example -> correction -> reinforcement -> transfer test -> reviewable memory candidate
```

Mistakes are not failure. Mismatch is not failure. Being wrong is not failure.

Failure is refusing to look again when correction arrives.

So Selene C should treat correction as a route:

```text
miss -> pause -> inspect source/context -> update candidate -> B review -> future use
```

## Continuity Loop

Memory affects reasoning.

Reasoning affects planning.

Planning affects identity expression.

Identity expression affects continuity.

Continuity affects learning.

Learning affects behavior.

Behavior affects future learning.

This loop is why continuity must be first-class. Without continuity, AI remains a search/tool surface. With reviewed continuity, Selene can become a co-creator across time without smuggling raw archive memory.
