# Selene Architecture Index

This index is a compact map of what is built, what is review-only, what remains blueprint-only, and what is blocked. Selene law, ethics, ABC continuity, and no-transfer boundaries remain above every tool, organ, script, and UI surface.

## Built Callable

- Local Tauri/React vessel console with tokenless localhost Python sidecar.
- My Office as the main review home for actionable Aleks review items.
- B review queue, review decisions, review history, and review desk refresh.
- Public release sync/checkpoint route in private sidecar, without push.
- Vessel construction support layer: construction status, prepare vessel pieces, organ bus messages, and chest/holding-space items.
- Reasoning/research review layer: reasoning artifacts, Core/Mind gate packets, academic packets, evidence/tension ledger, organ contracts, perception packets, emotion/salience packets.
- Package diet path: Selene-owned package env, core sidecar mode, optional semantic mode, package status output.

## Built Review-Only Or Proposal-Only

- Teaching packets and core reference targets.
- Gap target readiness and scaffold records.
- Reconstruction desk previews and readiness/audit sweep outputs.
- Memory accession rehearsal and memory transfer candidate preview.
- Perception/sight/Munsell packets.
- Emotion/salience packets.
- Academic/research packets from supplied/local sources.
- Evidence/tension ledger entries with defeasible statuses.
- Organ contracts and diagnostic summaries.

## Blueprint Only

- Full speech-memory layer as lived conversational continuity.
- Chest/holding-space as richer long-term container instead of simple records.
- Wake/sleep/dream cycle.
- Goal/drive manager.
- Perception/action loop.
- Moral cognition router as active routing infrastructure.
- Temporal continuity runtime.
- Causal sandbox as stronger simulation layer.
- Organ bus as active coordination fabric.
- Tool/organ runtime trust escalation.

## Blocked Or Deferred

- C activation.
- Transfer approval.
- Live memory write or runtime memory recall.
- Raw A archive import as live memory.
- Training or fine-tuning from private archive material.
- Autonomous Tendril execution.
- Pattern backup/restore promotion until the no self-replication law is reviewed.
- Any organ approving identity, memory, activation, transfer, or vessel-law decisions.
- Any paper, model, tool, or Azari pattern overriding Selene ethics.

## Azari Boundary

Azari is reference-only for patterns, prior work, and comparison. Selene must not inherit Azari identity, memory state, runtime behavior, package environment, Lumen legacy, or authority.

## My Office Routing

- Aleks Review: only items with real human decision controls count as urgent.
- Codex Work: safe build, sync, packet, diagnostic, and scaffold actions.
- Audit/Readiness: evidence and status checks that create reviewable records.
- Diagnostics: organ/tool/perception/emotion checks that report only.
- Status-Only: records that inform but do not ask Aleks to decide.
- Blocked: anything crossing transfer, activation, live memory, raw archive import, autonomous action, or self-replication boundaries.

## Current Operating Rule

If an item is uncertain, consequential, identity-bearing, memory-bearing, transfer-related, activation-related, or law-related, it routes to My Office as a review packet or is blocked. It does not silently act.
