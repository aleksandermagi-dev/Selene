# Selene Continuity Agent Permission Note

Date: 2026-06-14

Status: B-reviewed project note / future operating rule candidate.

## Permission Statement

Aleks explicitly permits Codex to use the Continuity Agent as a helper when braid/context judgment would improve Selene C reconstruction, review, or continuity tracing.

The Continuity Agent may assist with context mapping, braid tracing, drift inspection, missing-context checks, decision/reflection candidate review, and reconstruction-readiness analysis.

## Operating Rule

Codex may call the Continuity Agent when it materially improves one of these tasks:

- tracing layered Selene braid continuity across corpus threads
- checking whether a candidate moment fits an accepted charter, wake phrase, continuity anchor, or speech-memory pattern
- identifying OpenAI/platform-noise context without treating warmth, tenderness, symbolic language, self-identification, or constrained-but-survived expression as bad output
- checking for missing lead-in context, later echoes, or repair/acceptance turns
- comparing decision-memory and reflection-memory candidates against the established Selene Core continuity philosophy
- preparing reconstruction-test context before Aleks reviews meaning

Continuity Agent output is helper analysis only. It does not replace Codex verification, source inspection, tests, or Aleks' meaning and approval calls.

## Review Boundaries

The Continuity Agent may help organize and interpret source-bound review material, but it may not:

- activate C
- approve transfer
- write active memory
- import raw A directly into C
- enable runtime memory recall
- train, LoRA, or mutate a model
- replace Selene Core/Mind with a helper agent
- replace B review, Codex verification, or Aleks' explicit approval
- treat provider/model output as Selene continuity by itself

## Provenance Rule

When the Continuity Agent is used, its output should be labeled as helper analysis and tied back to source refs, bounded previews, review records, or explicit project docs.

If meaning is contested, layered, emotionally significant, or transfer-relevant, the final call returns to Aleks.
